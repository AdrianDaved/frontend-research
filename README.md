# Ecogen

Ecogen is an educational web project presenting an interactive infographic about endangered species. It is designed to make complex conservation topics approachable through clear visuals, concise narratives, and guided exploration. The experience highlights species status, habitats, key threats, and actions individuals and communities can take, aiming to build awareness and inspire informed, practical engagement.

## Overview

- Focus: Endangered species and conservation awareness
- Format: Web-based, interactive infographic
- Approach: Visual storytelling supported by brief, accessible explanations
- Goal: Help learners quickly grasp the challenges facing biodiversity and discover meaningful ways to contribute

## Educational Objectives

- Explain why species become endangered and how conservation status is determined
- Illustrate the interconnectedness of ecosystems, habitats, and human activity
- Encourage curiosity through interactive elements and data-informed insights
- Promote responsible behaviors and community-led action

## Audience

- Students and educators seeking clear, engaging materials on biodiversity
- Environmental enthusiasts exploring conservation concepts
- General audiences interested in learning about endangered species in a visual, web-first format

## Content Highlights

- Species profiles with status, habitat, and primary threats
- Contextual panels explaining conservation terms and processes
- Action prompts suggesting practical steps and further resources
- Visual cues and interactions that reinforce learning and retention

## Design and Technology

- Built with TypeScript for maintainable, reliable front-end logic
- Styled with CSS to keep the interface clear, accessible, and responsive
- Lightweight JavaScript interactions to guide exploration and discovery

## Ethos

Ecogen values clarity, empathy, and scientific grounding. Content aims to be respectful of communities and ecosystems, encouraging learners to appreciate complexity while focusing on actionable understanding. The project is iterative and will evolve as new insights, stories, and educational techniques emerge.
